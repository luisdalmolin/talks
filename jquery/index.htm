jQuery - Write less, do more

O que é?

É uma biblioteca javascript. "Write less, do more" é o 'lema'. Ou seja, escreva menos e faça mais.

Atualmente na versão 1.5.1.

Vantagens:

  • Tem uma comunidade muito ampla.
  • Curva de aprendizado pequena.
  • Cross-Browser, funciona em todos browsers utilizados hoje em dia. Até mesmo no IE6.

Algumas das maiores empresas do mundo estão usando jQuery, como Google, Dell, Mozilla, Wordpress e muito mais.

Diferença de sintaxe:

Javascript
var divs = documet.getElementByTagName(‘div’);
for( i = 0; i < divs.length; i++) {
    divs[i].style.display = ‘none’;
}

$(‘div’).hide();

Criação de plugins

A criação de plugins já exige um conhecimente um pouco mais avançado. Abaixo segue um simples exemplo.

(function($) {	
    $.fn.pluginteste = function(options) {
        var defaults = {
            altura  : 200,
            largura : 300,
            tempo   : 5000
        }		
        var opt  = $.extend( defaults, options );
        console.log('Altura: ' + opt.altura + ' | Largura: ' + opt.largura + ' | Tempo: ' + opt.tempo);	
	}
})(jQuery);

$('a').pluginteste({ altura : 250, tempo : 750 });

Pressione ctrl + shit + j para ver o console com o google chrome ou f12 com firefox.

Animações

Para exemplificar como podem ser feitas animações, segue como esta apresentação foi feita.

$(document).ready(function() {
   var
    x      = 1,
    $slide = $('.slide'),
    total  = $slide.lenght;
    
   // Esconde todos os slides
   $('.slide').hide();

   // Chama a função de exibir para começar a apresentação
   exibir( $('#slide-' + x) );

   $('.seta').click(function(e) { 
       e.preventDefault();
       var id = $(this).attr('id');

       if( id == 'seta-direita' ) {
           x++;
           if( x > total )
               x = 1;
       } else {
           x--;
           if( x < 1 )
               x = total;
       }
       $(this).attr('href', '#' + x);		
       exibir( $('#slide-' + x) );	
   });
});
